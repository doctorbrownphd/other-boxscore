Type a player name, team, stat category, or year. The synthesis engine translates pre-computed model outputs into plain-language findings.
How it works

The synthesis engine receives pre-computed data from the Color Line analysis pipeline and translates it into plain-language findings using Claude (Anthropic). Every numerical claim is grounded in the data visible in the raw data panel below each finding.

HIGH CONFIDENCEdirectly in the data
MODERATE CONFIDENCEminimal inference
CANDIDATErequires assumptions

Responses are cached for 24 hours. The same query returns the same synthesis.
